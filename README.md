# 📝 Web To-Do List App

A full-stack web-based To-Do List application built using Flask, SQLite, HTML, CSS, and JavaScript. The application allows users to manage daily tasks efficiently with features like task creation, completion tracking, deletion, search functionality, and dashboard statistics.

## 🚀 Features

* Add New Tasks
* View All Tasks
* Mark Tasks as Completed
* Delete Tasks
* Search Tasks
* Dashboard Statistics

  * Total Tasks
  * Completed Tasks
  * Pending Tasks
* SQLite Database Integration
* Responsive User Interface

## ▶️ Installation

```bash
git clone https://github.com/Deep-Kotak/web-todo-app.git

cd web-todo-app

pip install flask

python app.py
```

Open:

http://127.0.0.1:5000
